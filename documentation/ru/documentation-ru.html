Принцип использования


Для создания файла Excel на процедурном языке PL/pgSQL создается функция, которая выполняет следующее: определяет документ (pgxls.create), заполняет его (pgxls.add_row, pgxls.set_cell_value, pgxls.add_sheet), формирует и возвращает файл (pgxls.get_file).

Кол-во столбцов и их ширина определяются при создании документа или добавлении листа. Для каждого столбца можно переопределить стиль в зависимости от типа данных c помощью процедур pgxls.set_column_[alignment|border|fill|font|format]_[type].

Значение ячейки устанавливается процедурой pgxls.set_cell_value, которая копирует стиль из стиля столбца в зависимости от типа значения и имеет необязательные параметры для изменения любого свойства стиля. Также свойства стиля можно изменить отдельными процедурами pgxls.set_cell_[alignment|border|fill|font|format].

Для удобства использования определяется текущий столбец, который сбрасывается при добавлении строки и увеличивается при установке значения поля без указания столбца. При установке значения ячейки с указанием столбца текущий столбец переопределяется на указанный. При помощи процедур pgxls.set_column_current и pgxls.next_column можно изменять текущий столбец.

При добавлении строки или листа текущая строка записывается во временную таблицу в виде подготовленной части файла и не может быть изменена. Высота строки вычисляется автоматически, но может быть установлена вручную процедурой pgxls.set_row_height.

При создании больших файлов для заполнения ячеек необходимо использовать процедуры pgxls.set_cell_[type] и получать файл частями (pgxls.get_file_parts_query).

Для создания файла по SQL-запросу создана функция-обертка pgxls.get_file_by_query, а для добавления листа - процедура pgxls.add_sheet_by_query.

Не разрешается обращение к внутренним полям pgxls.xls и использование процедур и типов, начинающихся с подчеркивания "_"

pgxls.add_row


Процедура добавляет одну или несколько строк на лист, обнуляет текущий столбец

Параметры:
xls pgxls.xls-документ
count int-кол-во строк (необязательный, по умолчанию 1)

Примеры вызова:
call pgxls.add_row(xls);
call pgxls.add_row(xls, 5);

pgxls.add_sheet


Процедура добавляет новый лист в документ

Параметры:
xls pgxls.xls-документ
columns_widths int[]-ширины столбцов
columns_captions text[]-заголовки столбцов (необязательный)
sheet_name varchar-имя листа (необязательный)

Пример вызова:
call pgxls.add_sheet(xls, array[10,20], array['ID','Name'], 'Sheet2');

pgxls.add_sheet_by_query


Процедура добавляет новый лист с результатами SQL-запроса

Параметры:
xls pgxls.xls-документ
query text-SQL-запрос
sheet_name varchar-имя листа (необязательный)

Пример вызова:
call pgxls.add_sheet(xls, array[10,20], array['ID','Name'], 'Sheet2');

pgxls.alignment_horizontal


Тип enum, определяет выравнивание по горизонтали
Значения: left,center,right,justify,fill,distributed

pgxls.alignment_vertical


Тип enum, определяет выравнивание по вертикали
Значения: top,center,bottom,justify,distributed

pgxls.border_line


Тип enum, определяет внешний вид границы.
Значения: none,thin,thick,dashed,dotted,dashDot,dashDotDot,double

pgxls.clear_file_parts


Процедура удаляет файл из временной таблицы, используется совместно с pgxls.get_file_parts_query

Параметр:
xls pgxls.xls-документ

color$[name]


Функция возвращает цвет (тип varchar(6)) по названию

Параметры:
name-название: light_red,light_green,light_blue,dark_red,dark_green,dark_blue

Примеры вызова:
call pgxls.set_cell_font(xls, color=>pgxls.color$light_red());

pgxls.create


Функция создает, инициализирует и возращает документ (тип pgxls.xls)

Параметры:
columns_widths int[]-ширины столбцов
columns_captions text[]-заголовки столбцов (необязательный)
sheet_name varchar-имя листа (необязательный)

Пример вызова:
declare 
  xls pgxls.xls; 
begin
  xls := pgxls.create(array[10,20,30], sheet_name=>'My sheet');

pgxls.font_name$[family]


Функция возвращает имя (тип varchar) по семейству шрифта

Параметры:
family-семейство шрифта: sans,sans_serif,monospace

Примеры вызова:
call pgxls.set_column_font(xls, 1, name=>pgxls.font_name$sans_serif(), size=>16);
call pgxls.set_cell_value(xls, 123, font_name=>pgxls.font_name$monospace());

pgxls.get_column_name


Функция возвращает название столбца (тип varchar) по номеру

Параметры:

Примеры вызова:
column_name := pgxls.get_column_name(5);

pgxls.get_file


Функция формирует и возвращает файл (тип bytea)

Параметры:
xls pgxls.xls-документ

Пример вызова:
return pgxls.get_file(xls);

pgxls.get_file_by_query


Функция создает документ с данными по SQL-запросу, формирует и возвращает файл (тип bytea)

Параметры:
query text-SQL-запрос

Пример вызова:
select pgxls.get_file_by_query('select * from pg_class');

pgxls.get_file_parts_query


Функция формирует файл и возвращает SQL-запрос (тип varchar) для получения файла по частям. Используется при получении больших файлов, после вычитывания файла необходимо вызвать процедуру pgxls.clear_file_parts

Параметры:
xls pgxls.xls-документ

Пример вызова:
return query execute pgxls.get_file_parts_query(xls);

pgxls.get_format_code_boolean


Функция формирует логический код формата (тип varchar)

Параметры:
text_true varchar-текст для истинного значения (необязательный)
text_false varchar-текст для ложного значения (необязательный)
text_null varchar-текст для неопределенного значения (необязательный)

Примеры вызова:
format1 := pgxls.get_format_code_boolean('Yes', 'No', 'Null');
format2 := pgxls.get_format_code_boolean(text_true=>'1');

pgxls.get_format_code_numeric


Функция формирует числовой код формата (тип varchar)

Параметры:
decimal_places int-кол-во знаков после запятой
thousands_separated boolean-разделение тысяч (необязательный)

Примеры вызова:
format1 := pgxls.get_format_code_numeric(3);
format2 := pgxls.get_format_code_numeric(decimal_places=>0, thousands_separated=>true);

pgxls.set_cell_[type]


Процедура устанавливет значение ячейки, присваивает значение текущего столбца.
Используется при создании больших документов, т.к. не требуется проверка типа и изменение стиля. В остальных случая рекомендуется использовать pgxls.set_cell_value

Параметры:
type-тип данных: text,integer,numeric,date,time,timestamp,boolean
xls pgxls.xls-документ
value [type]-значение
column_ int-столбец (необязательный, по умолчанию следующий)

Примеры вызова:
call pgxls.set_cell_integer(xls, 123);
call pgxls.set_cell_numeric(xls, 123.4567, 2);
call pgxls.set_cell_text(xls, 'Hello');
call pgxls.set_cell_boolean(xls, false, column_=>4);

pgxls.set_cell_alignment


Процедура устанавливет выравнивание значения в ячейке

Параметры:
xls pgxls.xls-документ
column_ int-столбец
horizontal pgxls.alignment_horizontal-выравнивание по горизонтали (необязательный)
vertical pgxls.alignment_vertical-выравнивание по вертикали (необязательный)
indent int-отступ

Примеры вызова:
call pgxls.set_cell_alignment(xls, 'center', 'center', column_=>1);
call pgxls.set_cell_alignment(xls, vertical=>'top');
call pgxls.set_cell_alignment(xls, horizontal=>'justify', indent=>2);

pgxls.set_cell_border


Процедура устанавливет границу ячейки

Параметры:
xls pgxls.xls-документ
column_ int-столбец
around pgxls.border_line-граница всех сторон (необязательный)
left_ pgxls.border_line-граница слева (необязательный)
top pgxls.border_line-граница сверху (необязательный)
right_ pgxls.border_line-граница справа (необязательный)
bottom pgxls.border_line-граница снизу (необязательный)

Примеры вызова:
call pgxls.set_cell_border(xls, around=>'thin', column_=>1);
call pgxls.set_cell_border(xls, top=>'dashed', bottom=>'dashed');
call pgxls.set_cell_border(xls, left_=>'none');

Процедура устанавливет заливку ячейки

Параметры:
xls pgxls.xls-документ
column_ int-столбец
foreground_color varchar(6)-цвет в формате 'RRGGBB'

Примеры вызова:
call pgxls.set_cell_fill(xls, 'FF007700', column_=>1);
call pgxls.set_cell_fill(xls, foreground_color=>pgxls.color$dark_red());
call pgxls.set_cell_fill(xls, 'none');

pgxls.set_cell_font


Процедура устанавливет шрифт ячейки

Параметры:
xls pgxls.xls-документ
column_ int-столбец
name varchar-имя (необязательный)
size int-размер (необязательный)
bold boolean-жирный (необязательный)
italic boolean-курсив (необязательный)
underline boolean-подчеркнутый (необязательный)
strike boolean-зачеркнутый (необязательный)
color varchar(6)-цвет в формате 'RRGGBB' (необязательный)

Примеры вызова:
call pgxls.set_cell_font(xls, name=>'Times New Roman', bold=>true, column_=>1);
call pgxls.set_cell_font(xls, name=>pgxls.font_name$monospace(), italic=>false);
call pgxls.set_cell_font(xls, color=>'FF007700', underline=>true);

pgxls.set_cell_format


Процедура устанавливет формат ячейки

Параметры:
xls pgxls.xls-документ
code varchar-код формата
column_ int-столбец (необязательный)

Примеры вызова:
call pgxls.set_cell_format(xls, '0.000', column_=>1);
call pgxls.set_cell_format(xls, 'dd.mm.yyyy hh:mm');

call pgxls.set_cell_format(xls, thousands_separated=>true);
call pgxls.set_cell_format(xls, decimal_places=>2, thousands_separated=>false);
call pgxls.set_cell_format(xls, text_true=>'YES', text_false=>'NO', text_null=>'NULL');

pgxls.set_cell_value


Процедура устанавливет значение ячейки и позволяет менять параметры стиля, присваивает значение текущего столбца

Параметры:
xls pgxls.xls-документ
value anyelement-значение
column_ int-столбец (необязательный)
format_code varchar-код формата (необязательный)
font_name varchar-имя шрифта (необязательный)
font_size int-размер шрифта (необязательный)
font_bold boolean-жирный шрифт (необязательный)
font_italic boolean-шрифт курсивом (необязательный)
font_underline boolean-подчеркнутый шрифт (необязательный)
font_strike boolean-зачеркнутый шрифт (необязательный)
font_color varchar(6)-цвет шрифта в формате 'RRGGBB' (необязательный)
border_around pgxls.border_line-граница всех сторон (необязательный)
border_left pgxls.border_line-граница слева (необязательный)
border_top pgxls.border_line-граница сверху (необязательный)
border_right pgxls.border_line-граница справа (необязательный)
border_bottom pgxls.border_line-граница снизу (необязательный)
fill_foreground_color varchar(6)-цвет заливки в формате 'RRGGBB' (необязательный)
alignment_horizontal pgxls.alignment_horizontal-выравнивание по горизонтали (необязательный)
alignment_indent int-отступ при выравнивании (необязательный)
alignment_vertical pgxls.alignment_vertical-выравнивание по вертикали (необязательный)

Примеры вызова:
call pgxls.set_cell_value(xls, 123);
call pgxls.set_cell_value(xls, now(), 2, format_code=>'dd.mm.yyyy hh:mm');
call pgxls.set_cell_value(xls, 'Hello'::text, font_size=>10, font_bold=>true, fill_foreground_color=>'FF007700');
call pgxls.set_cell_value(xls, true, border_around=>'thin', alignment_horizontal=>'left', alignment_indent=>2);

pgxls.set_column_alignment_[type]


Процедура устанавливет выравнивание значения в ячейках столбца в зависимости от типа данных. Если тип данных не указан (процедура pgxls.set_column_alignment), то выравнивание устанавливается для всех типов. Выравнивание по горизонтали определяется типом pgxls.alignment_horizontal, по вертикали - pgxls.alignment_vertical

Параметры:
type-тип данных: text,integer,numeric,date,time,timestamp,boolean (необязательный)
xls pgxls.xls-документ
column_ int-столбец
horizontal pgxls.alignment_horizontal-выравнивание по горизонтали (необязательный)
indent int-отступvertical pgxls.alignment_vertical

Примеры вызова:
call pgxls.set_column_alignment(xls, 1, 'center', 'center');
call pgxls.set_column_alignment_integer(xls, 2, vertical=>'top');
call pgxls.set_column_alignment_text(xls, 3, horizontal=>'justify', indent=>2);

pgxls.set_column_border_[type]


Процедура устанавливет границу ячеек столбца в зависимости от типа данных. Если тип данных не указан (процедура pgxls.set_column_border), то граница устанавливается для всех типов. Линия границы (внешний вид) определяется типом pgxls.set_border_line, для отмены границы используется линия 'none'

Параметры:
type-тип данных: text,integer,numeric,date,time,timestamp,boolean (необязательный)
xls pgxls.xls-документ
column_ int-столбец
around pgxls.border_line-граница всех сторон (необязательный)
left_ pgxls.border_line-граница слева (необязательный)
top pgxls.border_line-граница сверху (необязательный)
right_ pgxls.border_line-граница справа (необязательный)
bottom pgxls.border_line-граница снизу (необязательный)

Примеры вызова:
call pgxls.set_column_border(xls, 1, around=>'thin');
call pgxls.set_column_border_integer(xls, 2, top=>'dashed', bottom=>'dashed');
call pgxls.set_column_border_date(xls, 3, left_=>'none');

type-тип данных: text,integer,numeric,date,time,timestamp,boolean
xls pgxls.xls-документ

Примеры вызова:

call pgxls.set_column_format_boolean(xls, 12, text_true=>'YES', text_false=>'NO', text_null=>'NULL');
