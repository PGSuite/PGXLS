Usage method


PL/pgSQL function is created that does: created document (pgxls.create), adds rows and sheets, sets document cells (pgxls.add_row, pgxls.add_sheet), sets cells (pgxls.set_cell_value), builds and returns file (pgxls.get_file).

Column count and their width are determined when creating a document or adding a sheet. For each column, can override style depending on data type using procedures pgxls.set_column_[alignment|border|fill|font|format]_[type].

Cell value set using procedure pgxls.set_cell_value, which copies style from column style depending on value type and has optional parameters to change any style property. Also style properties can be changed by separate procedures pgxls.set_cell_[alignment|border|fill|font|format].

For ease of use, the current column is determined, which is reset when a row is added and incremented when setting a field value without specifying a column. Procedures pgxls.set_column_current and pgxls.next_column change current column explicitly.

When adding row or sheet, current row written to temporary table and cannot be changed. Row height calculated automatically, but can be set manually using procedure pgxls.set_row_height.

When creating large files, must use procedures pgxls.set_cell_[type] for sets cells and get file in parts (pgxls.get_file_parts_query).

There are functions pgxls.get_file_by_query and pgxls.add_sheet_by_query to get file and add sheet by SQL query.

Not allowed to access internal fields of pgxls.xls and use procedures and types that start with underscore "_"

pgxls.add_row


Procedure adds one or more rows to sheet, sets current column to zero

Parameters:
xls pgxls.xls-document
count int-row count (optional, default 1)

Examples:
call pgxls.add_row(xls);
call pgxls.add_row(xls, 5);

pgxls.add_sheet


Procedure adds new sheet to document

Parameters:
xls pgxls.xls-document
columns_widths int[]-columns widths
columns_captions text[]-columns captions (optional)
sheet_name varchar-sheet name (optional)

Example:
call pgxls.add_sheet(xls, array[10,20], array['ID','Name'], 'Sheet2');

pgxls.add_sheet_by_query


Procedure adds new sheet with SQL query data

Parameters:
xls pgxls.xls-document
query text-SQL query
sheet_name varchar-sheet name (optional)

Example:
call pgxls.add_sheet(xls, array[10,20], array['ID','Name'], 'Sheet2');

pgxls.alignment_horizontal


Enum type, defines horizontal alignment
Values: left,center,right,justify,fill,distributed

pgxls.alignment_vertical


Enum type, defines vertical alignment
Values: top,center,bottom,justify,distributed

pgxls.border_line


Enum type, defines border line
Values: none,thin,thick,dashed,dotted,dashDot,dashDotDot,double

pgxls.clear_file_parts


Procedure removes file from temporary table, used together with pgxls.get_file_parts_query

Parameter:
xls pgxls.xls-document

color$[name]


Function returns color (type varchar(6)) by name

Parameters:
name-name: light_red,light_green,light_blue,dark_red,dark_green,dark_blue

Examples:
call pgxls.set_cell_font(xls, color=>pgxls.color$light_red());

pgxls.create


Function creates, initializes and returns document (type pgxls.xls)

Parameters:
columns_widths int[]-columns widths
columns_captions text[]-columns captions (optional)
sheet_name varchar-sheet name (optional)

Example:
declare 
  xls pgxls.xls; 
begin
  xls := pgxls.create(array[10,20,30], sheet_name=>'My sheet');

pgxls.font_name$[family]


Function returns name (type varchar) by font family

Parameters:
family-font family: sans,sans_serif,monospace

Examples:
call pgxls.set_column_font(xls, 1, name=>pgxls.font_name$sans_serif(), size=>16);
call pgxls.set_cell_value(xls, 123, font_name=>pgxls.font_name$monospace());

pgxls.get_column_name


Function returns column name (type varchar) by number

Parameters:
column_ int-column

Examples:
column_name := pgxls.get_column_name(5);

pgxls.get_file


Function builds and returns file (type bytea)

Parameters:
xls pgxls.xls-document

Example:
return pgxls.get_file(xls);

pgxls.get_file_by_query


Function creates document by SQL query, builds and returns file (type bytea)

Parameters:
query text-SQL query

Example:
select pgxls.get_file_by_query('select * from pg_class');

pgxls.get_file_parts_query


Function builds file and returns SQL query (varchar type) to get file in parts. Used for large files, after receiving file, need to call procedure pgxls.clear_file_parts

Parameters:
xls pgxls.xls-document

Example:
return query execute pgxls.get_file_parts_query(xls);

pgxls.get_format_code_boolean


Function builds logical format code (varchar type)

Parameters:
text_true varchar-text for true value (optional)
text_false varchar-text for false value (optional)
text_null varchar-text for null value (optional)

Examples:
format1 := pgxls.get_format_code_boolean('Yes', 'No', 'Null');
format2 := pgxls.get_format_code_boolean(text_true=>'1');

pgxls.get_format_code_numeric


Function builds numeric format code (varchar type)

Parameters:
decimal_places int-decimal places
thousands_separated boolean-thousands separated (optional)

Examples:
format1 := pgxls.get_format_code_numeric(3);
format2 := pgxls.get_format_code_numeric(decimal_places=>0, thousands_separated=>true);

pgxls.merge_cells


Procedure merges cells

Parameters:
xls pgxls.xls-document
column_count int-column count (optional, default 1)
row_count int-row count (optional, default 1)
column_ int-column from which the merge begins (optional, default current)

Examples:
call pgxls.merge_cells(xls, 5);
call pgxls.merge_cells(xls, row_count=>5);
call pgxls.merge_cells(xls, 4, 4, 6);

pgxls.set_cell_[type]


Procedure sets cell value, assigns current column.
Used for large documents (no type checking and no style change) In other cases, recommended to use pgxls.set_cell_value

Parameters:
type-data type: text,integer,numeric,date,time,timestamp,boolean
xls pgxls.xls-document
value [type]-value
column_ int-column (optional, default next)

Examples:
call pgxls.set_cell_integer(xls, 123);
call pgxls.set_cell_numeric(xls, 123.4567, 2);
call pgxls.set_cell_text(xls, 'Hello');
call pgxls.set_cell_boolean(xls, false, column_=>4);

Parameters:

Examples:

pgxls.set_cell_value


Procedure sets cell value and allows to change style parameters, assigns current column

Parameters:
xls pgxls.xls-document
value anyelement-value
column_ int-column (optional)
format_code varchar-format code (optional)
font_name varchar-font name (optional)
font_size int-font size (optional)
font_bold boolean-bold font (optional)
font_italic boolean-italic font (optional)
font_underline boolean-underline font (optional)
font_strike boolean-strike font (optional)
font_color varchar(6)-font color in format 'RRGGBB' (optional)
border_around pgxls.border_line-border around (optional)
border_left pgxls.border_line-border left (optional)
border_top pgxls.border_line-border top (optional)
border_right pgxls.border_line-border right (optional)
border_bottom pgxls.border_line-border bottom (optional)
fill_foreground_color varchar(6)-fill color in format 'RRGGBB' (optional)
alignment_horizontal pgxls.alignment_horizontal-horizontal alignment (optional)
alignment_indent int-alignment indent (optional)
alignment_vertical pgxls.alignment_vertical-vertical alignment (optional)

Examples:
call pgxls.set_cell_value(xls, 123);
call pgxls.set_cell_value(xls, now(), 2, format_code=>'dd.mm.yyyy hh:mm');
call pgxls.set_cell_value(xls, 'Hello'::text, font_size=>10, font_bold=>true, fill_foreground_color=>'FF007700');
call pgxls.set_cell_value(xls, true, border_around=>'thin', alignment_horizontal=>'left', alignment_indent=>2);

pgxls.set_column_alignment_[type]


Procedure sets alignment of values in cells of column depending on data type. If data type not specified (procedure pgxls.set_column_alignment), then alignment set for all types. Horizontal alignment determined by type pgxls.alignment_horizontal, vertical - pgxls.alignment_vertical

Parameters:
type-data type: text,integer,numeric,date,time,timestamp,boolean (optional)
xls pgxls.xls-document
column_ int-column
horizontal pgxls.alignment_horizontal-horizontal alignment (optional)
indent int-alignment indent
vertical pgxls.alignment_vertical-vertical alignment (optional)

Examples:
call pgxls.set_column_alignment(xls, 1, 'center', 'center');
call pgxls.set_column_alignment_integer(xls, 2, vertical=>'top');
call pgxls.set_column_alignment_text(xls, 3, horizontal=>'justify', indent=>2);

Procedure sets border cells of column depending on data type. If data type not specified (procedure pgxls.set_column_border), then border set for all types. Border line determined by type pgxls.set_border_line, to cancel border uses 'none'

Parameters:
type-data type: text,integer,numeric,date,time,timestamp,boolean (optional)
xls pgxls.xls-document
column_ int-column
around pgxls.border_line-border around (optional)
left_ pgxls.border_line-border left (optional)
top pgxls.border_line-border top (optional)
right_ pgxls.border_line-border right (optional)
bottom pgxls.border_line-border bottom (optional)

Examples:
call pgxls.set_column_border(xls, 1, around=>'thin');
call pgxls.set_column_border_integer(xls, 2, top=>'dashed', bottom=>'dashed');
call pgxls.set_column_border_date(xls, 3, left_=>'none');

pgxls.set_column_format_[type]


Procedure sets format cells of column depending on data type.

For ease of use, available procedures for integer, numeric and boolean data types:
pgxls.set_column_format_integer(inout xls pgxls.xls, column_ int, thousands_separated boolean default true)
pgxls.set_column_format_numeric(inout xls pgxls.xls, column_ int, decimal_places int default 2, thousands_separated boolean default true)
pgxls.set_column_format_boolean(inout xls pgxls.xls, column_ int, text_true varchar default 'True', text_false varchar default 'False', text_null varchar default '')
Parameters:
type-data type: text,integer,numeric,date,time,timestamp,boolean
xls pgxls.xls-document
column_ int-column
code varchar-format code

Examples:
call pgxls.set_column_format_numeric(xls, 1, '0.000');
call pgxls.set_column_format_timestamp(xls, 2, 'dd.mm.yyyy hh:mm');

call pgxls.set_column_format_integer(xls, 10, thousands_separated=>true);
call pgxls.set_column_format_numeric(xls, 11, decimal_places=>2, thousands_separated=>false);
call pgxls.set_column_format_boolean(xls, 12, text_true=>'YES', text_false=>'NO', text_null=>'NULL');
